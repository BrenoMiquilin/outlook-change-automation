# 🐍 Python Outlook Change Automation

## 📌 Overview
Python script that automates email responses based on change management data.

## ⚙️ How it works
- Captures data from Microsoft Planner (title and date)
- Searches related emails in Outlook
- Generates a formatted HTML response
- Sends or drafts a reply automatically

## 🛠️ Technologies
- Python
- pyautogui
- pyperclip
- win32com (Outlook COM)

## 🎯 Results
- Reduced manual email writing
- Standardized communication
- Increased efficiency in change processes

## 🧠 Key Concepts
- Automation
- Outlook integration (COM)
- Data extraction
- HTML generation
